== Description ==

Awesome Instagram Feed is a very lighweight plugin used to create charming galleries of Instagram images.You can have two layout.You can either showcase instagram images into slider layout or grid layout.Use fully responsive and mobile friendly plugin to attract your websiteâ€™s audience in a flash.You can add the instagram photos into your site for brand and self-promotion.

How to Work with the Plugin ==

1. Upload `awesome-instagram-feed.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [awesome_instagram_feed] in your pages and go to customizer/Awesome Instagram feed Option for other various options

== Shortcodes with Attributes ==

[awesome_instagram_feed feedlayout="slider"] //For slider Layout
[awesome_instagram_feed feedlayout="grid"] //For Grid Layout
